## Overview of My Work
I have been working on various projects that demonstrate my skills and interests in backend development, DevOps, and full-stack development. Below is an overview of my key projects:

### Current Projects
- **Talk2U Academy App**: A comprehensive language learning platform designed to teach multiple languages (English, Spanish, and French) through an interactive and gamified experience. This app features user authentication, course management, gamification, and a subscription model. The technologies used include Next.js, TypeScript, Firebase, React Native, and Expo. It leverages Firebase for deployment, hosting, and functions, and uses microservices for AI, including Gemini and OpenAI integrations.
- **Talk2U Preview**: A trial version of English classes available at [Talk2U Preview](https://testegratis.talk2uacademy.com.br/). Both this and the Talk2U Academy App leverage AI technologies like Gemini and OpenAI.

### Recent Projects
1. **Movies API**: A project to learn and implement NestJS for building efficient server-side applications. Technologies: TypeScript, TypeORM, Swagger, PostgreSQL, Docker, Redis. This project emphasizes Docker for containerization.
2. **Activities Tracker**: An application to track various activities with a structured MVC pattern using modern web development practices. Technologies: TypeScript, NestJS, TypeORM, Docker.
3. **WABA-NOX**: A Heroku app designed to consume Meta API for WhatsApp Business, aimed at logistics companies.
4. **Scrapper Web**: A web scraper built with asynchronous Python, leveraging threads and mutexes to ensure efficient data scraping.
5. **Thread-Pygame**: A game development project using threading for handling concurrent programming tasks.
6. **Arduino-NTC**: A hardware-related project involving Arduino, showcasing skills in IoT and embedded systems programming.
7. **CI/CD & Automation**: Projects focused on continuous integration and delivery, utilizing tools like Jenkins, GitHub Actions, and Docker.
8. **DevOps & DevSecOps**: Initiatives to integrate security into the DevOps process, leveraging tools like Grafana, Prometheus, and various CI/CD pipelines.
9. **Real-time Chat App**: A chat application with features like typing indicators and chat history sharing. Built with Node.js, Socket.io, and React.
10. **E-commerce Platform**: A full-stack e-commerce application with features like user authentication, product management, and order processing. Built with MERN stack.
